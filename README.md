# Datos
En esta actividad se creó la estructura inicial del repositorio Ayudant-a-S.A, incorporando los archivos esenciales para la gestión de información de la empresa. Se generaron dos archivos principales: clientes.csv, que contiene el registro básico de clientes, y ventas.csv, que almacena las ventas realizadas con sus datos correspondientes. Además, se corrigió el nombre del archivo de ventas, se actualizaron los contenidos con datos estructurados y se dejó el repositorio organizado para futuras ampliaciones o análisis. Con esto, el proyecto queda preparado para continuar con la integración de nuevas funcionalidades y mejoras.
